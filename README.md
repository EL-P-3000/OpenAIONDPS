# Open AION DPS

このツールは、日本語版 The Tower of AION のDPSを計測するツールです。

This is a DPS tool for The Tower of AION Japanese edition.

このツールを利用するには、日本語版 The Tower of AION の設定を変更して、Chat.logにログを出力する必要があります。
